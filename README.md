# Kinetopia — Marketing Site

The marketing website for **Kinetopia**, a one-time-purchase iOS fitness tracker that runs entirely on-device.

> The fitness tracker without the subscription.

## Live site

Hosted on GitHub Pages at <https://scwill12.github.io/kinetopia-website>.

## Stack

- Single static HTML file, no build step.
- No third-party fonts (system serif stack).
- No analytics, no tracking, no JavaScript dependencies — matches the privacy-first ethos of the app.

## Editing

Open `index.html` in any browser to preview locally. Edit the file, commit, push — GitHub Pages publishes the change within a minute.
